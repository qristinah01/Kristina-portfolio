- `hero.png` — Full-width hero image (16:9 or 16:10). The densest, most impressive overview shot.

## Tips
- Export at 2x for retina. Target ~2400px wide for landscape, ~1200px wide for portrait/mobile.
- Use consistent padding/framing across all exports.
- If a screen doesn't exist yet, skip it — the page will show a gray placeholder with the filename.
