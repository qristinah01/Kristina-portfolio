# Drop real project visuals here

Replace the placeholder blocks on case study pages by dropping images into this folder,
then updating the relevant project in `lib/projects.ts`.

Recommended naming:
- synon-hero.jpg, synon-detail-1.jpg, etc.
- halo-hero.jpg, halo-detail-1.jpg, etc.

**Compress to ~200–400 KB each before upload.** Next.js will further optimize them
into AVIF/WebP automatically.
